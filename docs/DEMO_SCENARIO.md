# Сценарий демонстрации StudioBooking MVP
Общее время: ~10–12 минут  
Цель: Показать полный user flow от регистрации до бронирования и админ-панели.

## 1. Главная страница (30 сек)
- Открыть frontend: http://localhost:3000
- Показать красивый лендинг
- Подсветить кнопки "Войти" и "Зарегистрироваться"
- Кратко: "Это главная страница, где пользователи видят каталог студий и могут быстро начать"

## 2. Регистрация клиента (1 мин)
- Нажать "Зарегистрироваться" → выбрать "Клиент"
- Заполнить форму:
    - Email: testclient@demo.kz
    - Пароль: demo123
    - Имя: Асем
    - Телефон: +77771234567
- Нажать "Зарегистрироваться"
- Показать: "Успешно! Токен сохранён в localStorage, пользователь перенаправлен в каталог"
- Переход на главную с авторизованным состоянием (иконка профиля)

## 3. Каталог студий (2 мин)
- Показать список студий (5–7 штук из seed)
- Применить фильтр "Город: Алматы"
- Применить фильтр "Тип комнаты: Fashion"
- Показать, как работает пагинация (если >10 студий)
- Кликнуть на "Light Studio Pro" → перейти в детали

## 4. Детальная страница студии (1 мин)
- Показать: описание, рейтинг, отзывы, фото, телефон, адрес
- Прокрутить до комнат: показать 2–3 зала с ценами и фото
- Показать отзывы (2–3 штуки)
- Кнопка "Забронировать" → перейти к календарю

## 5. Бронирование (2–3 мин)
- Выбрать комнату "Белый зал"
- Показать календарь доступности (GET /rooms/:id/availability)
- Выбрать время: завтра 14:00–16:00
- Заполнить заметки: "Фотосессия для портфолио"
- Нажать "Забронировать"
- Показать подтверждение: "Бронирование создано! Ожидайте подтверждения от владельца"

## 6. Мои бронирования (30 сек)
- Перейти в профиль → "Мои бронирования"
- Показать список (1–2 брони)
- Подсветить статус "pending"

## 7. Вход как владелец студии (1 мин)
- Выйти → войти: owner1@studio.kz / owner123
- Показать: "Теперь я владелец"

## 8. Панель владельца (2 мин)
- Перейти в "Мои студии" → увидеть свои студии
- Открыть студию → увидеть список бронирований (GET /studios/:id/bookings)
- Подтвердить бронирование: PATCH /bookings/:id/status → "confirmed"
- Отметить оплату: PATCH /bookings/:id/payment → "paid"
- Показать уведомления (колокольчик): "Новое бронирование", "Верифицировано"

## 9. Вход как админ (1–2 мин)
- Выйти → войти: admin@photostudio.kz / admin123
- Перейти в админ-панель
- Показать статистику (GET /admin/statistics)
- Показать pending студии (если есть)
- Верифицировать студию (POST /admin/studios/:id/verify)

## 10. Заключение (30 сек)
- "Это полный цикл: от регистрации клиента до управления бронированиями и админ-контроля"
- "Backend полностью готов к интеграции с фронтендом"
